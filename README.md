# Pokedex
Trilha JS Developer - Pokedex

# Pré-requisitos
- Nodejs
- npm
- Instalar os pacotes:
  - `npm install -g http-server` 
  - Teste: `http-server`

## Iniciar o http
`http-server ./`

- Na página, desabilitar o cache: Acessar a porta (localhost:8080) > F12 > Network > Disable cache
